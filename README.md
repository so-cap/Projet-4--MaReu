# MaReu
**School project**

Maréu est une application de gestion de réunions.

Cette application permet :
- De planifier des réunions
- D'afficher la liste des réunions
- De les filtrer par date et heure 
- De voir chacune des réunions en détail
- Et de les supprimer.
